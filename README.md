# PUM - PLUS ULTRA MANAGER

_a lightweight mod manager for My Hero Ultra Rumble_

![image](https://github.com/Bacrian/PUM/blob/aa5cf3135c591c9eb3224d9f9b0eb700b58fb4f4/pum-logo.png)

_Based on Python_

PUM is a mod loader and mod manager for My Hero Ultra Rumble, that uses a simple system to check folders and copy files right into the game.
![image](https://i.imgur.com/DrZocV4.png)

**WARNING!**

Before using PUM make sure your ~mods file folder is empty, otherwise ALL mods within the folder will get removed.

# How to Use
1. Open PlusUltraManager.exe and select Game Path
2. Search for your MHUR Paks folder and apply.
3. To install mods, open the mods folder or use the Open Mods Folder button in the executable and paste your .PAKs
